Getting started

Here's how to get rolling:

  1. Use bin/rails generate to creat your
    models and controllers

    2. To see all available options, run it without parameters

  2. Set up a route to replace this page

    3. You're seeing this page because your're running in
    development mode and you haven't set a root route yet.

    Routes are set up in config/routes.rb

  3. Configure your database

    4. if you're not using SQLite (the default), edit
    config/database.ymlwith your username and password
